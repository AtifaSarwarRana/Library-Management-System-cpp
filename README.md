# Library Management System (C++)

A console-based Library Management System developed in C++ using Object-Oriented Programming (OOP) and File Handling concepts.

## Features

- Add Book
- Display Books
- Search Book by ID
- Issue Book
- Return Book
- Update Book Information
- Delete Book
- Unique Book ID Validation
- Save Data to File
- Load Data from File

## Technologies Used

- C++
- OOP (Classes and Objects)
- File Handling
- Arrays
- Functions

## Project Structure

- `Library Management System.cpp` - Main source code
- `books.txt` - Stores book records

## Concepts Implemented

- Classes and Objects
- Functions
- Arrays of Objects
- File Handling (ifstream / ofstream)
- CRUD Operations
- Data Validation

## How to Run

1. Compile the source code:

```bash
g++ "Library Management System.cpp" -o library
```

2. Run the program:

```bash
./library
```
